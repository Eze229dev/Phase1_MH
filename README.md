## 🔧 Customization Tips

### Change the calculator fee rates:
Open `index.html`, search for `0.10` and `0.01` in the JavaScript section. Adjust as needed.

### Change the colors:
Search for hex codes:
- `#10b981` (main green)
- `#06b6d4` (accent cyan)

### Add your logo:
Add an `<img>` tag at the top of the hero section

### Add email capture:
Replace the "Get Started" button with a form

See `DOCUMENTATION.md` for more details.
